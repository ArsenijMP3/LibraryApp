# LibraryApp — Модули 2 и 3 (Задания 2.2–2.5, 3.1–3.4)

Десктоп-приложение на C# / WinForms (.NET 10), подключение к Access (`base.accdb`) через OLE DB.
**Модуль 2:**
- **2.2 Оформление и стиль** — единый стиль окон, акцентный цвет #2E8B57, нейминг PascalCase/camelCase.
- **2.3 Авторизация** — окно входа первое (`LoginForm`), вход по логину/паролю, гостевой доступ, ФИО+роль в правом верхнем углу, кнопка «Выход».
- **2.4 Список записей** — каталог книг с раскраской строк (красный/зелёный/голубой).

**Модуль 3:**
- **3.1** — модальные окна (`ShowDialog`) дают последовательную навигацию, кнопка «Назад» в форме добавления/редактирования, все исключения перехватываются и показываются через `MessageBox` с заголовком и пиктограммой.
- **3.2** — поиск по названию/автору/жанру/издательству, фильтр по жанру («Все жанры» первым элементом), сортировка по году издания (возр./убыв.) — всё в реальном времени (`TextChanged`/`SelectedIndexChanged`, без кнопок), работает совместно, сортировка сохраняется. Доступно **библиотекарю и администратору**, скрыто для гостя/читателя.
- **3.3** — форма добавления/редактирования (`BookEditForm`) в отдельном модальном окне: открывается по кнопке «Добавить» или двойным кликом по строке; числовые поля (`NumericUpDown`) без отрицательных значений; ID — авто (MAX+1, превью), при редактировании неизменяем; фото приводится к 300×200 px, путь сохраняется в БД, старое фото удаляется при замене; модальность `ShowDialog` исключает открытие двух окон редактирования одновременно.
- **3.4** — удаление с проверкой связанных выдач (`CountRelatedLoans`) — запрещает удаление, если есть хоть одна связанная запись в `Выдачи`; список автообновляется после добавления/редактирования/удаления.
Доступ к CRUD — только **администратор**.

## Как запустить
1. Установите **.NET SDK** (у вас 10.0.301 — подходит, `TargetFramework` уже `net10.0-windows`).
2. Если на машине не установлен Access/Office — поставьте **Microsoft Access Database Engine 2016 Redistributable** (64-разрядную версию, если SDK x64).
3. Скопируйте ваш файл `base.accdb` в корень проекта (рядом с `LibraryApp.csproj`).
4. В терминале, в папке проекта:
   ```bash
   dotnet run
   ```
5. Логины для проверки ролей (из `Скрипт_заполнения.sql`):
   - `sidorova` / `pass456` — читатель (только просмотр)
   - `kuznecov` / `lib789` — библиотекарь (+ поиск/фильтр/сортировка)
   - `morozova` / `admin123` — администратор (+ добавление/редактирование/удаление)
   - кнопка «Войти как гость» — только просмотр
